## 서비스 소개
<div align="left">

> 개발 기간: 2026. 03. ~ 2026. 05.

**SafeSign: Safe + Sign**

AI 분석으로 확보하는 안전한 계약

임대차 계약 당사자가 서명 전 문서를 업로드하여 잠재적 위험을 점검하며,

관련 법령에 기반한 독소 조항 검출과 맞춤형 특약 추천을 통해 부당한 피해를 예방하는 AI 계약서 분석 웹 서비스
